## Markdown 
Repositories op GitHub zetten automatisch **Markdown**-code om naar **HTML**. 
 
In Markdown heb je b.v. een *blockquote* dat in de letterlijke zin v.h. woord bedoeld is om *quotes* van personen op te maken.
